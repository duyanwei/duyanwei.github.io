BIOGRAPHY

I'm a visual SLAM engineer at HoloMatic. Inc. I'm involved in multiple tasks, including sensor extrinsic and intrisnic calibration, lidar mapping, visual feature based mapping and localization, monocular visual inertial odometry.

My research interests include Simultaneous Localization and Mapping(SLAM), Visual Inertial Odometry(VIO), Stereo Visual Odometry, Robotics and Computer Vision.

PROJECT

Title: GPS-aided visual mapping

Visual mapping with GPS/IMU sensor, techniques including feature detection and matching, triangulation, global bundle adjustment, keyframe selection and map management.

Title: Quadrotor path planning and trajectory generation

Given known occupied 3D map, implement an algorithm to plan an obstacle-free path that connects the start and goal position, and generate feasible trajectory for the quadrotor to move with certain amount of time.

Title: Gesture Recognition from IMU data with Hidden-Markov Model

Using IMU raw readings that are collected from a mobile device to train a Hidden Markov Model to recognize different arm motion gestures by moving the mobile device.

Title: 2D Lidar SLAM with Partical Filter

Given readings from wheel encoder and lidar data sampled from a mobile robot when moving in a building. The goal is to build a map as well as to localize the robot itself in the map.
